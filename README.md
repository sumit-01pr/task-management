

# 1️⃣ Task Management Application


A full-stack MERN Task Management Application that allows users to create, update, delete, and manage tasks efficiently with secure JWT-based authentication.

## 🚀 Live Demo

Frontend: https://your-vercel-link.vercel.app

Backend: https://task-management-o87x.onrender.com

## 📖 Features

- User Registration & Login
- JWT Authentication
- Create Tasks
- Update Tasks
- Delete Tasks
- Mark Tasks as Completed/Pending
- Protected Routes
- Responsive Design

## 🛠️ Tech Stack

### Frontend
- React.js
- React Router
- Axios
- CSS3

### Backend
- Node.js
- Express.js

### Database
- MongoDB Atlas

### Authentication
- JWT
